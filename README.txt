## Descripción

Este proyecto consiste en una página web que permite a los usuarios agregar albums o canciones a mi banda favorita, "No Te Va Gustar".  Este proyecto tiene la intención de crear una experiencia de escucha de música similar a Spotify en función de su elección.

## Funcionalidades Principales
	
- Registro de usuarios
- Inicio de sesión
- Creación de albums y canciones en base a la banda ya creada
- Reproducción de canciones
- Interfaz de usuario atractiva y fácil de usar
- Diseño responsive


## Tecnologías Utilizadas

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- Node.js
- Express.js
- MongoDB y Mongoose
- Autenticación con JWT
- Bcrypt para encriptación de contraseñas
